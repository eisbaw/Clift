## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
seL4 verified pointer arithmetic correctness (array indexing, pointer comparison, pointer-to-member). TASK-0106 (Done) added array subscript support to CImporter, but no task covers proving that array accesses are within bounds as part of the UB-freedom proofs. Need: (1) Array bounds guard generation: CImporter emits guard(0 <= i && i < len) for arr[i], (2) Prove bounds guards are satisfiable (no spurious UB), (3) Pointer arithmetic model: p + n is valid iff p..p+n*sizeof within same allocation, (4) Example: verify a function with array iteration, proving all accesses in bounds. This is critical for any real C code -- array bounds are the most common source of UB in practice.
